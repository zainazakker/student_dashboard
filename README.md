Student Recruitment Dashboard

Project Overview
This is a simple frontend web application built using HTML, CSS, and JavaScript.

The project helps manage student details in a basic dashboard without using any backend or database.

It allows users to:
- Add student details such as name, email, course interest, and status
- View all added students in a table
- Search students by course interest
- Store data in the browser using localStorage so data stays after refresh

Technologies Used
- HTML
- CSS
- JavaScript

Tools Used
- ChatGPT (for help with understanding code, structure, and debugging)

What I learned
- Creating and handling form inputs
- Storing and retrieving data using localStorage
- Filtering data using JavaScript
- Basic CRUD logic in frontend

How to Run
1. Download or open the project folder
2. Open index.html in a browser
3. No installation or server required
